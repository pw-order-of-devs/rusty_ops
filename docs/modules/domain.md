# `domain` library

[< Back to modules README](README.md)

## Design Purpose:

`domain` is a shared library providing the domain models for the `RustyOps` system. It contains the definitions of structures used in the system:
- agents
- auth:
  - credentials
  - permissions
  - resources
  - roles
  - users
- commons:
  - search filters
- jobs
- pipelines
- projects
- templates:
  - pipeline template
